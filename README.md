# 🏓 Pong Game in C++

This repository contains a simple **Pong Game** developed in **C++** using the **SDL2** library.

## 📌 About

This project was completed as part of my **Week 5 & Week 6 Internship Task** at **InternPe**.

The objective of the task was to develop a Pong game in C++ while applying object-oriented programming concepts, collision detection, and basic game development techniques.

## ✨ Features

- Paddle movement using keyboard controls
- Ball collision detection
- Score tracking
- Ball speed acceleration after collisions
- Ball reset when missed
- Simple and clean SDL2 graphics

## 🛠️ Technologies Used

- C++
- SDL2
- CMake

## 🎮 Controls

- **W** → Move Paddle Up
- **S** → Move Paddle Down
- **Esc** → Exit Game
